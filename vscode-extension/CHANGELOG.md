# Changelog

All notable changes to the "vscode-timeline-extractor" extension will be documented in this file.

## [0.1.0] - 2026-01-28

### Added

- Initial release
- Browse all files in VS Code's timeline history
- View all versions of any file
- Reconstruct directories with latest file versions
- Time travel - reconstruct directories at a specific timestamp
- Compare versions with current file using diff view
- Restore any historical version
- Export individual file versions
- Activity bar with dedicated views
- Context menu integration in Explorer
- Configuration options for custom timeline paths
- Metadata export for reconstructed directories
